Prowlarr

Prowlarr হলো একটি indexer manager যা Radarr, Sonarr, Bazarr এর সাথে কাজ করে। এটি বিভিন্ন টরেন্ট/Usenet indexer connect করতে দেয় এবং সব ম্যানেজমেন্ট এক জায়গায় আনা যায়।

Prerequisites

  • Ubuntu 20.04 / 22.04 Server
  • Docker & Docker Compose installed
  • Radarr/Sonarr আগে install করা থাকলে integration সহজ হবে

1 Install Prowlarr (Docker)

mkdir -p ~/docker/prowlarr
cd ~/docker/prowlarr

cat > docker-compose.yml <<EOF
version: "3.8"
services:
  prowlarr:
    image: lscr.io/linuxserver/prowlarr:latest
    container_name: prowlarr
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=Asia/Dhaka
    volumes:
      - ./config:/config
    ports:
      - 9696:9696
    restart: unless-stopped
EOF

docker compose up -d

2 Access UI

URL: http://SERVER_IP:9696

3 Radarr/Sonarr Integration

Prowlarr UI > Settings > Apps এ গিয়ে Radarr বা Sonarr যোগ করুন।
API Key use করতে হবে যেটা Radarr/Sonarr settings থেকে নেয়া যাবে।
